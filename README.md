# 📈 Stock Price Prediction API (LSTM + FastAPI)

---

## 🚀 Description

A **Machine Learning-based Stock Prediction API** built using **LSTM (Long Short-Term Memory)** and **FastAPI**.

This project fetches historical stock data, processes it using deep learning, and provides:
- 📊 Price Predictions  
- 📈 EMA Indicators  
- 📉 Trend Analysis  

---

## ✨ Features

- 📊 Stock price prediction using LSTM  
- ⚡ FastAPI backend (high performance)  
- 📈 EMA indicators (20, 50, 100, 200)   
- 📉 Trend detection (UP 📈 / DOWN 📉)  
- 🌐 Easy frontend integration  
- 🧠 Data preprocessing using MinMaxScaler  

---

## 🛠️ Tech Stack

- **Python**
- **FastAPI**
- **TensorFlow / Keras**
- **NumPy & Pandas**
- **Scikit-learn**
- **yFinance**
